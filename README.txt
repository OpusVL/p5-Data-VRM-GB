British VRM Dating Module

This exports a function decode_vrm which takes a vehicle registration mark (e.g. 'AA01 AAA')
as its argument and returns a hashref containing start_date and end_date (both DateTime objects).


API status: Unstable - we haven't fully decided on the API yet
